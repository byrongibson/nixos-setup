
Quick setup
git@github.com:byrongibson/nixos-setup.git

Get started by creating a new file or uploading an existing file. We recommend every repository include a README, LICENSE, and .gitignore.
https://github.com/byrongibson/nixos-setup/new/main
https://github.com/byrongibson/nixos-setup/upload

…or create a new repository on the command line

echo "# nixos-setup" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:byrongibson/nixos-setup.git
git push -u origin main

…or push an existing repository from the command line

git remote add origin git@github.com:byrongibson/nixos-setup.git
git branch -M main
git push -u origin main
